# 产品想法

**产品名称**

ResearchFlash（研究灵感速记本）

**产品用途**

 本项目旨在解决科研人员高认知负荷下的短期记忆溢出问题。它不是笔记软件，而是一个允许你『放心遗忘』的中转站。核心机制是Capture (瞬间捕捉灵感) -> Sleep (潜伏) -> Resurface (灵感重现)。

**4个重要的现实约束**

  - 首屏必须直接展示输入框，无登录或引导页拦截
  - 单条灵感限制280字以内
  - 每周日晚8点推送本周灵感回顾
  - 必须支持离线写入，网络恢复后自动同步

**涉及到的认知模型**
  本系统包括以下关键实体：

  - flash：灵感碎片（核心实体）
  - user：使用者
